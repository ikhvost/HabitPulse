# Habit(b) pulse

### Installation
`npm ci --include=dev`

### Start
- With platform selection: `npm run start`
- Android only: `npm run android`
- iOS only: `npm run ios`
- Web only: `npm run web`

### Test
- Run: `npm run test`
- Run in watch mode: `npm run test:watch`

### Linter
- Check: `npm run lint`
- Fix: `npm run lint:fix`
