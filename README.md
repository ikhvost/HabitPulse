## ğŸš€ Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm
- React Native development environment setup

### Installation

```bash
npm ci --include=dev
```

### Run with platform selection

```bash
npm run start
```

#### Android specific

```bash
npm run android
```

#### iOS specific

```bash
npm run ios
```

#### Web specific

```bash
npm run web
```

### Development

#### Run tests

```bash
npm run test
```

#### Run tests in watch mode

```bash
npm run test:watch
```

#### Check linting

```bash
npm run lint
```
